# Complete i18n Conversion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert all hardcoded English text in packages/views to i18n using existing `packages/core/i18n/` infrastructure.

**Architecture:** Extend existing i18n modules on-demand, use `useLocale()` hook in components, update `types.ts` for new dictionary keys.

**Tech Stack:** React Context (LocaleProvider), Zustand-free translation system, js-cookie for locale persistence.

- [ ] **Step 2: Create localized formatActivity function**

Replace the `formatActivity` function with a version that uses translations:

```tsx
function formatActivity(
  entry: TimelineEntry,
  t: DashboardDict,
  resolveActorName?: (type: string, id: string) => string,
): string {
  const details = (entry.details ?? {}) as Record<string, string>;
  switch (entry.action) {
    case "created":
      return t.issues.timeline.createdIssue;
    case "status_changed":
      return `${t.issues.timeline.changedStatusFrom} ${statusLabel(details.from ?? "?")} ${t.common.to} ${statusLabel(details.to ?? "?")}`;
    case "priority_changed":
      return `${t.issues.timeline.changedPriorityFrom} ${priorityLabel(details.from ?? "?")} ${t.common.to} ${priorityLabel(details.to ?? "?")}`;
    case "assignee_changed": {
      const isSelfAssign = details.to_type === entry.actor_type && details.to_id === entry.actor_id;
      if (isSelfAssign) return t.issues.timeline.selfAssigned;
      const toName = details.to_id && details.to_type && resolveActorName
        ? resolveActorName(details.to_type, details.to_id)
        : null;
      if (toName) return `${t.issues.timeline.assignedTo} ${toName}`;
      if (details.from_id && !details.to_id) return t.issues.timeline.removedAssignee;
      return t.issues.timeline.changedAssignee;
    }
    case "due_date_changed": {
      if (!details.to) return t.issues.timeline.removedDueDate;
      const formatted = new Date(details.to).toLocaleDateString("en-US", { month: "short", day: "numeric" });
      return `${t.issues.timeline.setDueDateTo} ${formatted}`;
    }
    case "title_changed":
      return `${t.issues.timeline.renamedFromTo} "${details.from ?? "?"}" ${t.common.to} "${details.to ?? "?"}"`;
    case "description_updated":
      return t.issues.timeline.updatedDescription;
    case "task_completed":
      return t.issues.timeline.completedTask;
    case "task_failed":
      return t.issues.timeline.taskFailed;
    default:
      return entry.action ?? "";
  }
}
```
